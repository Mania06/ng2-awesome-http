# ng2-awesome-http

## Installation

To install this library, run:

```bash
$ npm install ng2-awesome-http --save
```

## Development

To generate all `*.js`, `*.js.map` and `*.d.ts` files:

```bash
$ npm run build
```

To lint all `*.ts` files:

```bash
$ npm run lint
```
